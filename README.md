# 롤 스킬 피하기 게임

Firebase 랭킹 연결이 적용된 버전입니다.

## 실행 방법

`index.html` 파일을 더블클릭하거나 GitHub / Vercel에 업로드하면 실행됩니다.

## 조작 방법

- WASD / 방향키: 이동
- Space: 기본 대시
- Q: 존야의 모래시계
- E: 벨트
- F: 유체화
- ESC: 일시정지
- 상점에서 1~3: 구매
- 상점에서 4 또는 Enter: 나가기

## Firebase 설정

이 파일에는 Firebase 설정값이 이미 들어가 있습니다.

Firestore Database를 만들고 Rules만 설정하면 온라인 랭킹이 작동합니다.
