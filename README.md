# 롤 스킬 피하기 게임

HTML5 Canvas로 만든 롤 스킬 피하기 스타일 미니게임입니다.

## 실행 방법

`index.html` 파일을 더블클릭하면 바로 실행됩니다.

## 조작 방법

- WASD / 방향키: 이동
- Space: 기본 대시
- Q: 존야의 모래시계
- E: 벨트
- F: 유체화
- ESC: 일시정지
- 일시정지에서 1 또는 Enter: 계속하기
- 일시정지에서 2: 다시하기
- 상점에서 1~3: 구매
- 상점에서 4 또는 Enter: 나가기

## 주요 기능

- 60FPS 고정으로 모니터 주사율이 달라도 속도 동일
- 3000점마다 랜덤 상점 등장
- 상점 종료 후 3-2-1 카운트다운
- 다시하기 후 3-2-1 카운트다운
- 스페이스 대시는 깜빡임 대신 파란 잔상/파동 효과
- 시간이 지날수록 난이도 상승
- 랜덤 증강 20개 이상
- 코인, 속도 증가, 체력 회복 아이템 드랍
- 존야의 모래시계, 벨트, 유체화 구매 가능
- 일시정지 메뉴와 다시하기 기능
