# 롤 스킬 피하기 게임

Firebase 랭킹 연결 + PC/모바일 조작 지원 버전입니다.

## 실행 방법

`index.html` 파일을 더블클릭하거나 GitHub / Vercel에 업로드하면 실행됩니다.

## PC 조작

- WASD / 방향키: 이동
- Space: 기본 대시
- Q: 존야의 모래시계
- E: 벨트
- F: 유체화
- ESC: 일시정지
- 상점에서 1~3: 구매
- 상점에서 4 또는 Enter: 나가기

## 모바일 조작

- 왼쪽 가상 조이스틱: 이동
- 오른쪽 큰 버튼: 대시
- Q 버튼: 존야의 모래시계 / 상점 1번 선택
- E 버튼: 벨트 / 상점 2번 선택
- F 버튼: 유체화 / 상점 3번 선택
- 오른쪽 위 Ⅱ 버튼: 일시정지
- 상점 화면에서는 상점 선택지를 직접 터치해서 구매 가능

## 주요 기능

- 시작 화면에서 이름 입력
- 게임 오버 시 Firebase 랭킹 자동 등록
- 랭킹 TOP 10 표시
- Firebase 연결 실패 시 localStorage 로컬 랭킹 사용
- 60FPS 고정
- 3000점마다 랜덤 상점 등장
- 상점 종료 후 3-2-1 카운트다운
- 스페이스/대시 파란 잔상 효과
- 랜덤 증강 20개 이상
